Flexbox Row

Three properties to use to create a Flexbox layout:

 .row {
 display:flex;
 flex-direction:row;
 flex-wrap:wrap;
 }

Flexbox Column

This will create a Flexbox column layout:

 .column {
 display: flex:
 flex-direction: column
  }

CSS Grid Layout

A tool to help you build a 12-column layout using CSS Grid:

 .grid {
 display: grid
 width: 100%
 grid-template-columns:

  }
